# Спринт 10: Групповой проект. YaMDb и API для него.
### Описание
"API для Yatube" реализованный в рамках обучения на курсе проэкт социальной сети
в которой есть возможность работать с базой данных не только через представление в HTML-страниц,
но и посредством описанного API. Реализованна возможность создавать, удалять, комментировать записи,
а так же подписаться на или отписаться от определённого автора.
### Технологии
Python 3.7, Django 2.2, DRF, JWT
